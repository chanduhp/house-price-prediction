# House Price Prediction

## Project Overview
This project predicts house prices using Machine Learning with Linear Regression.

## Dataset
- Boston Housing Dataset
- Total Records: 506
- Features: 13
- Target: House Price (MEDV)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model
- Linear Regression

## Model Performance
- Mean Squared Error (MSE): 24.29
- R² Score: 0.6688
